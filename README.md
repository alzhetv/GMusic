## Local development

### Local project

Clone the repository:
```bash
git clone https://github.com/Gecolay/GMusic.git
```

### Build

Run the `gradle build` command.

The final `GMusic-x.x-x.jar` file will be in the [`build/libs`](./build/libs) folder.
